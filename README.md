# StarWars-API

Project to list and manage Star Wars movies using data from [SWAPI](https://swapi.dev/).

## Prerequisites

Before you begin, ensure you have the following tools installed on your machine:
- Git
- JDK 17 (or higher versions)
- Maven

## Cloning the Repository

To obtain a local copy of the project for development and testing purposes, follow these steps:

```bash
# Clone this repository
$ git clone https://github.com/marciodc/starwar-spring-boot-api.git

# Go to the project folder in your terminal/cmd
$ cd starwar-spring-boot-api
```
